HUD Manager - это небольшая утилита для создания локального зеркала файлов HUD. Предназначена для совместного использования с SRC Repair.

Данный продукт может работать в двух режимах: анонимном и авторизованном. Вы сможете выбрать предпочитаемый режим, а также ввести свой токен GitHub API на заключительном шаге установки.

HUD Manager поддерживает следующие способы хранения API-токена:
  * скрипт запуска (токен будет сохранён непосредственно внутри автоматически сгенерированного скрипта) — используется по умолчанию;
  * переменные окружения (токен будет сохранён в переменных окружения текущего пользователя системы) — при его использовании по завершении установки потребуется перезагрузка;
  * ручной режим - токен не будет сохраняться, но может быть передан при помощи переменных окружения.

Персональный токен GitHub API можно сгенерировать здесь: https://docs.github.com/en/articles/creating-a-personal-access-token-for-the-command-line.
